# scribdStrip

Chrome extension for revealing pages content without subscribing.

# Use

Just click the 'S' button on chrome bar to reveal the pages in front of you.

![alt text](./src/assets/images/demo.gif "Demo gif")

# Supported sites

 - scribd.com/docs
 - studocu.com
